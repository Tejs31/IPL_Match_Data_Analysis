# 🏏 IPL Match Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an IPL match dataset using **Python**. The objective is to uncover meaningful insights into team performance, toss impact, player statistics, and venue trends through data cleaning, feature engineering, and visualization.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing.
* Explore team and player performance.
* Analyze toss decisions and match outcomes.
* Identify venue-wise trends.
* Visualize key insights using charts and graphs.

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📊 Key Analysis

* Conducted end-to-end Exploratory Data Analysis (EDA).
* Answered **11 business questions** using data analysis techniques.
* Performed data cleaning by handling missing values and correcting data types.
* Engineered features using Python lambda functions.
* Used **groupby**, aggregations, filtering, and sorting for analysis.
* Created **8+ visualizations** to communicate insights effectively.

---

## 📈 Key Insights

* Winning the toss showed an approximate **50% correlation** with winning the match.
* Identified the highest-performing teams and players.
* Analyzed venue-wise match frequency and outcomes.
* Explored batting and bowling performance trends across seasons.
